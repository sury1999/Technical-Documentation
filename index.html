Basic rule syntax

The syntax for each specified value depends on the data type defined for each specified property.

Be aware that any syntax error in a rule definition invalidates the entire rule. Invalid rules are ignored by the browser. Note that CSS rule definitions are entirely (ASCII) text-based, whereas DOM-CSS / CSSOM (the rule management system) is object-based.

background-image: url("../../media/examples/lizard.png"); //for setting background images padding: 10px 50px 20px; //for adding padding to elements color: #00ff00; //for setting text-color

Selectors

Basic selectors are fundamental selectors; these are the most basic selectors that are frequently combined to create other, more complex selectors.

Selector list A, B Specifies that both A and B elements are selected. This is a grouping method to select several matching elements.

Combinators are selectors that establish a relationship between two or more simple selectors, such as "A is a child of B" or "A is adjacent to B."

/* The element with id="demo" */
#demo {
border: red 2px solid;
}
 /* Paragraphs that come immediately after any image */
img + p {
font-weight: bold;
}
